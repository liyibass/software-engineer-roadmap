# AWS 雲端基礎建設課程大綱

> **核心理念**：你在公司聽到 VPC、EKS、T3 這些詞，這門課就是為此而生的。
> 從「雲端是什麼」開始，一路走到你能看懂公司的 VPC 架構圖、理解 EKS 叢集的運作方式、讀懂 T3 執行個體規格單、知道為什麼帳單突然暴增。
> 這是一本獨立的書，但建議先有基礎課程（`lessons/basic/`）Part 7 部署與課外讀物 E-13 規模化的概念，再進來把概念全部落地到 AWS 的真實服務。

---

## 這門課的定位

| | 說明 |
|---|------|
| **前置知識** | 基礎課程 Part 7（部署）、課外讀物 E-11（快取）、E-13（規模化與 K8s 概念） |
| **適合對象** | 已經會寫全端、想看懂公司雲端架構、準備踏入 DevOps / SRE 的人 |
| **學完你能** | 讀懂公司的 VPC / EKS 架構圖、做部署環境選型、看懂 AWS 帳單、用 IaC 管理基礎設施 |

> 共用的通用知識（Docker、Git、網路、安全）仍在頂層 `課外讀物/`，這門課會在需要時引用。

---

## 學習路徑總覽

```
Part 1 雲端基礎 → Part 2 IAM → Part 3 EC2 → Part 4 VPC（核心）
   → Part 5 儲存 → Part 6 受管服務 → Part 7 容器平台 ECS/EKS
   → Part 8 Serverless → Part 9 CI/CD 與 IaC → Part 10 觀測與成本
```

> Part 4（VPC）是理解「公司雲端架構」最核心的一章，後面的容器、受管服務都建立在它之上。

## Part 4 — 網路 VPC

> 這是理解「公司雲端架構」最核心的章節。

### 章節列表

- `aws-4-1` VPC 是什麼？為什麼需要在 AWS 裡建立自己的私有網路

- `aws-4-4` Internet Gateway vs NAT Gateway：誰能主動出門？誰能被外部拜訪？
- `aws-4-5` Security Group vs NACL：Instance 層 vs Subnet 層的防火牆差在哪

- `aws-4-7` Multi-AZ 架構：為什麼要把資源分散到不同機房
- `aws-4-8` 看懂 VPC 架構圖：從一張圖讀懂公司的雲端網路全貌

## Part 6 — 受管服務

### 章節列表

- `aws-6-3` Application Load Balancer（ALB）：流量怎麼分配、健康檢查怎麼運作

## Part 7 — 容器平台 ECS / EKS

### 章節列表

- `aws-7-1` ECR：Docker Image 的倉庫（AWS 版的 Docker Hub）

- `aws-7-7` 看懂公司的 EKS 架構：真實場景全景拆解（結合 VPC + ALB + RDS）
