## 概念說明

> 🎈 這是趣味章節，輕鬆讀。但這個故事是整個科技業過去半世紀的引擎。

### 邏輯閘是用什麼做的？電晶體

前面幾章一直用邏輯閘（[cs-2-1]）、正反器（[cs-2-4]）。但它們本身又是用什麼做的？答案是——**電晶體（transistor）**。

電晶體是現代電子的基石，本質上是一個**「用電來控制的開關」**：

```
普通開關：要用手去撥。
電晶體：  用「一個電訊號」去控制「另一條電路通不通」——不用手，用電控制。
         給它訊號 → 通（1）；不給 → 斷（0）。
```

這個「用電控制開關」的能力，正是組成邏輯閘所需要的。比喻：

```
幾個電晶體，依特定方式接線 → 組成一個邏輯閘（AND/OR/NOT）
很多邏輯閘 → 組成加法器、正反器（cs-2-3、2-4）
很多這些 → 組成 CPU
```

所以一路往下追：**電腦 = 大量電晶體的組合**。一顆現代 CPU 裡有**數百億個電晶體**，全部是這種微小的「電控開關」。

### 摩爾定律：每兩年塞進兩倍

1965 年，Intel 創辦人之一 **戈登·摩爾（Gordon Moore）** 提出一個觀察，後人稱為**摩爾定律（Moore's Law）**：

> **積體電路上能容納的電晶體數量，大約每兩年就會翻一倍。**

```mermaid
graph LR
    Y1["1970s<br/>幾千個電晶體"] --> Y2["1990s<br/>數百萬個"] --> Y3["2010s<br/>數十億個"] --> Y4["今天<br/>數百億個"]
```

這張圖在說：在同樣大小的晶片上，電晶體數量呈「指數成長」——每隔約兩年翻倍。注意摩爾定律不是物理定律，而是一個「**觀察與預測**」，但它驚人地準確維持了數十年。

### 為什麼這讓電腦越來越快又便宜？

電晶體越塞越多、越做越小，帶來連鎖好處：

- **更快**：電晶體越小，電訊號跑的距離越短，切換越快。
- **更便宜**：同樣一片晶片塞進更多運算能力，「每單位運算」的成本暴跌。
- **更省電、更小**：所以才有今天輕薄又強大的手機、筆電。

這就是為什麼「**今天一支普通手機的運算力，遠超 30 年前的超級電腦**」（呼應 [cs-0-3]）。摩爾定律是過去半世紀「科技產品不斷變強又變便宜」背後的引擎——它讓昨天的奢侈品變成今天的日常。

### 但是……它正在遇到瓶頸

摩爾定律不可能永遠持續，因為**電晶體不可能無限縮小**——當尺寸接近「幾個原子」的大小，會遇到物理極限（量子效應、散熱問題）。近年電晶體微縮的速度確實在放緩。

於是業界轉向別的方法追求效能：

```
不能再靠「電晶體更小」時，改用其他招：
   多核心：一顆晶片放多個 CPU 核心（cs-5-3 並行）
   專用晶片：為 AI、繪圖設計專門的處理器（GPU、TPU）
   新材料、新架構：研究取代矽的可能
```

所以「電腦怎麼繼續變快」，從「單純縮小電晶體」轉向「更聰明的架構設計」——這也是為什麼「並行」「多核心」（[cs-5-3]、rust 課程 Part 8）變得越來越重要。

## 範例：指數成長有多誇張

```
「每兩年翻倍」聽起來普通，但指數成長很可怕：
   翻 1 次：2 倍
   翻 5 次（約 10 年）：32 倍
   翻 10 次（約 20 年）：1024 倍
   翻 15 次（約 30 年）：超過 3 萬倍

→ 這就是為什麼幾十年下來，電腦效能成長了「百萬倍」級別。
  指數成長，是人類直覺最容易低估的力量。
```
